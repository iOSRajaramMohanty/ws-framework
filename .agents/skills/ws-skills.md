# Skills

## Skill: Build Adapter
- Use adapter pattern
- Keep interface generic
- Example: RedisAdapter, KafkaAdapter

## Skill: Build Middleware
- Express-style chaining
- Async support
- Error propagation

## Skill: Event System
- Map<string, handlers[]>
- Efficient dispatch
- Avoid blocking operations

## Skill: Scalability
- Avoid shared memory assumptions
- Design for distributed systems

## Skill: Plugin Design
- Plugins should not modify core directly
- Use hooks/events
